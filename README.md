# 🌐 Personal Portfolio

A modern and responsive personal portfolio website developed using React, Vite, and TypeScript to showcase my skills, projects, and experience.

## 🚀 Features

- Responsive design for all devices
- Interactive and modern UI
- About Me section
- Skills showcase
- Project portfolio
- Contact section
- Smooth animations and navigation

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
portfolio/
├── public/
├── src/
│   ├── assets/
│   ├── components/

│   ├── App.tsx
│   └── main.tsx
├── package.json

```

## ▶️ Getting Started

1. Clone this repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL displayed in the terminal.
